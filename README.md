# 🎓 Student Performance Predictor

This project is a machine learning-based system that predicts student academic performance using various features such as study time, attendance, and previous grades.

---

## 🚀 Features

- 📊 Data analysis using real-world dataset  
- 🧹 Data preprocessing (handling missing values, normalization, outliers)  
- 📈 Exploratory Data Analysis (EDA) to identify key trends  
- 🤖 Model training using:
  - Linear Regression  
  - Decision Tree  
  - K-Nearest Neighbors (KNN)  
- 📉 Model evaluation using RMSE, MAE, and R²  

---

## 🧠 Workflow

1. Load dataset  
2. Perform data cleaning and preprocessing  
3. Conduct EDA to understand relationships  
4. Train multiple machine learning models  
5. Compare performance and select best model  

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- Jupyter Notebook
